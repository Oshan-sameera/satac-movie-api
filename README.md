### How to run locally

```bash


# to run the frontend 
cd frontend
npm install # to install node modules
npm start 

# to run the backend
cd backend 
mvn package #Build Spring Boot Project with Maven 
java -jar target/movies-api-0.0.1-SNAPSHOT.jar # Run Spring Boot app with java -jar command


```
